DeadLine Support

Need help with DeadLine? Contact support and include enough detail so we can reproduce and fix quickly.

Contact

Email: manay.mcpserver@gmail.com

What to Include in a Support Request

  • Device model and OS version (Android/iOS).
  • App version (from Settings > About).
  • Integration involved (for example: Notion, Jira, Google Calendar).
  • What happened vs what you expected.
  • Screenshots/screen recording if available.

Common Issues

  • Integration not syncing: disconnect and reconnect that integration, then run sync.
  • OAuth callback issues: retry from a stable network and ensure app is updated to latest build.
  • Ads/consent issues: use in-app privacy choices where available and retry.

Response Time

Typical response target: within 3 business days.

Security Reports

For security issues, email the same address and use subject line: [Security] DeadLine report.
